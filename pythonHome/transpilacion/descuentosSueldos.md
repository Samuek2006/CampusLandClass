## Descripción:
Este script calcula el sueldo neto de una persona tras aplicar tres descuentos fijos al sueldo base.

**Variables:**

sueldo: sueldo base ingresado por el usuario
desc1, desc2, desc3: porcentajes de descuento fijos (0.01, 0.04, 0.005)
res1, res2, res3: montos descontados individualmente
resf: sueldo neto después de los descuentos

*Entrada ficticia:*
sueldo = 1000000
Prueba de escritorio paso a paso:
Calcular descuento 1: res1 = 1000000 * 0.01 = 10000
Calcular descuento 2: res2 = 1000000 * 0.04 = 40000
Calcular descuento 3: res3 = 1000000 * 0.005 = 5000
Calcular sueldo final: resf = 1000000 - (10000 + 40000 + 5000) = 945000

*Resultado:*
El sueldo neto es: 945000
