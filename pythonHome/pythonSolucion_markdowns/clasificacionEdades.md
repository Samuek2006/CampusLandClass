# clasificacion Edades

## Descripción breve

Este programa realiza operaciones relacionadas con **clasificacion edades**.

## Prueba de escritorio

Este programa no recibe entradas del usuario.

**Paso a paso:**
1. El programa no requiere entradas, inicia con valores internos.
2. Se ejecutan las operaciones según el flujo del código.
3. Se imprime el resultado final en pantalla.

## Salida esperada

El resultado final es el calculado con los datos ficticios dados.
