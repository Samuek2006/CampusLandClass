## Descripción:
Este script clasifica un número según su divisibilidad entre 2, 3 o ambos.

**Variables:**

- numero: número ingresado por el usuario
- clasificacion: resultado según las reglas de divisibilidad

*Entrada ficticia:*
numero = 18

**Prueba de escritorio paso a paso:**

*Se evalúa si el número es divisible entre 2 y 3:*
18 % 2 = 0 → divisible entre 2
18 % 3 = 0 → divisible entre 3
→ divisible entre ambos

*Resultado:*
El número 18 es divisible entre 2 y 3
