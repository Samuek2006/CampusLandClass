## Descripción:
Este script permite registrar tres notas de un estudiante, calcular el promedio y mostrar si aprobó o no.

**Variables:**

- nota1, nota2, nota3: calificaciones individuales
- promedio: promedio de las tres notas
- resultado: indica si aprobó (promedio ≥ 3.0)

*Entrada ficticia:*
nota1 = 4.0, nota2 = 2.5, nota3 = 3.5

**Prueba de escritorio paso a paso:**

*Se calcula el promedio:*
promedio = (4.0 + 2.5 + 3.5) / 3 = 10.0 / 3 ≈ 3.33
→ promedio ≥ 3.0 → aprobado

*Resultado:*
El estudiante tiene un promedio de **3.33** → **Aprobado**
