## Descripción:
Este script determina si un año es bisiesto o no, según las reglas del calendario gregoriano.

**Variables:**

- año: el año ingresado por el usuario
- esBisiesto: valor lógico o resultado textual indicando si el año es bisiesto

*Entrada ficticia:*
año = 2024

**Prueba de escritorio paso a paso:**

*Se evalúa la condición de bisiesto:*
2024 % 4 = 0
2024 % 100 ≠ 0 → cumple condición → **es bisiesto**

*Resultado:*
El año 2024 **es bisiesto**
