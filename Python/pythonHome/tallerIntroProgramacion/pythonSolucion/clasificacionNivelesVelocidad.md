## Descripción:
Este script clasifica el nivel de velocidad de un vehículo según la cantidad de km/h ingresados.

**Variables:**

- velocidad: velocidad del vehículo en kilómetros por hora
- clasificacion: nivel asociado (baja, moderada, alta, peligrosa)

*Entrada ficticia:*
velocidad = 95

**Prueba de escritorio paso a paso:**

*Se evalúa la velocidad:*
95 → entre 81 y 100 → clasificación: alta

*Resultado:*
Una velocidad de 95 km/h se clasifica como **alta**
